# RatioTrack

RatioTrack is a free online financial ratio calculator designed to help users quickly analyze a company's financial performance without using spreadsheets.

## 🚀 Features

- Calculate liquidity, profitability, and debt ratios
- Simple and fast interface
- No Excel or accounting knowledge required

## 🌐 Live Website

https://ratiotrack.com

## 🎯 Purpose

This project was built to make financial analysis more accessible for entrepreneurs, students, and anyone interested in understanding financial data.

## 🛠️ Tech

- HTML
- CSS
- JavaScript
